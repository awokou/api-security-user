## Requirements

01) Java 21
02) Maven 3.X
03) Spring Boot 3.X
04) Spring Data JPA
05) Spring Web
06) Spring security
07) PostGreSql Database

## Project setup

01) Clone the project

    	git clone https://github.com/awokou/api-security-user.git

02) Clean and build the project using maven

    	- mvn clean install
        - mvn spring-boot:run
